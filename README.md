# Bot de Gastos Personales - Toto

## Cómo subir a Railway

1. Creá cuenta en railway.app
2. Click en "New Project" → "Deploy from GitHub"
3. Subí estos archivos a un repo de GitHub o usá "Deploy from local"
4. Agregá la variable de entorno: BOT_TOKEN = tu_token

## Comandos del bot

- Mandar "café 800" → se registra y categoriza
- /resumen → gastos del mes con gráfico
- /hoy → gastos de hoy
- /categorias → desglose por categoría
- /consejos → análisis y tips de ahorro
- /borrar_ultimo → borra el último gasto

## Categorías automáticas

- 🍔 Comida (almuerzo, café, delivery...)
- 🍽️ Salidas (cena, restaurant, amigos...)
- 💃 Salidas nocturnas (boliche, tragos, chicas...)
- 👗 Ropa
- ⚽ Fútbol
- 🚗 Transporte (nafta, uber...)
- 📱 Servicios (Netflix, Spotify...)
- 💳 Deudas (cuotas, tarjeta...)
- 🏠 Casa (supermercado, expensas...)
